Instructions


Redeeming a License

  • Start by navigating to the Manage page. 
  • First you will need to add a computer by filling out the add computer form.  Enter a display Name for your new computer.  Download the Authority FX 'Mac Address Finder' python script.  Run the script and select a mac address that will be used to identify your computer.  Paste the mac address into the field and click the 'Add Computer' button.
  • Now you will assign licenses to your computer.  Use the Computer drop-down to select your computer. Then use the License drop-down to select the plugin license that you wish to assign.  Click the 'Assign License' button.
  • Once you have assigned the desired plugins to your computer, you then need to look back at the 'Computers' table where you will see "Redeem" in the Action field.  Clicking 'Redeem' will generate and serve a license file for download; however it will also lock all licenses assigned to that computer.  This action cannot be undone without emailing plugins@authorityfx.com, so please ensure that all information and assignments are correct before you redeem your license.  Follow the instructions on the 'Downloads' page to install your license file and plugins.

Lost or Corrupted License File

  • If you need a replacement license file for a computer, simply navigate to the Manage page and click 'Redeem' to generate a new license.

Adding Additional Plugins to an Existing Computer

  • Adding additional licenses to an existing computer is easy.  Simply assign your new licenses to the existing computer and redeem a new license file.  The new license file will include all licenses assigned to the computer for which it was redeemed.  All you need to do is replace your old ‘afx-license.dat’ file with the new one.  Refer to the ‘Downloads’ page for the proper location of your license file.

Hardware Change

  • If you’ve upgraded to a new computer please email plugins@authorityfx.com and request to have your license unlocked from the current computer.  You will need to provide us with your login email and the computer id of the computer you wish to unlock.
